# The Care Ranch — Brochure Feedback Project

## What this is
A static feedback document for the client's brochure (version 7), styled in the Care Ranch design language and hosted via GitHub Pages.

## Files
- `index.html` — the styled feedback page (self-contained, no build step)
- `Feedback-Brochure-v7.md` — the same feedback as plain markdown
- `Client-input/` — original .docx brochure from the client

## Deployment
- **Repo:** https://github.com/willem4130/care-ranch-brochure-feedback
- **Live URL:** https://willem4130.github.io/care-ranch-brochure-feedback/
- To update: edit `index.html`, commit, and push. GitHub Pages rebuilds automatically.

## Context
- The brochure was written in Dutch first, then translated/composed into English — be mild on grammar/language feedback, frame it as "worth a native English speaker's pass"
